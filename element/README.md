# Element Web Profile

CallChat Community uses Element Web as a Matrix-compatible web client profile.

Recommended approach:

1. Download a known Element Web release from the official project.
2. Keep upstream license files and notices.
3. Place `config.sample.json` as the basis for `config.json`.
4. Add CallChat theme assets without modifying upstream source where possible.
5. Pin the release in production.
6. Apply security updates after testing.

Do not hide upstream attribution or distribute closed-source forks without licensing review.
