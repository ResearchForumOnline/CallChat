# CallChat and Matrix

CallChat is not trying to break Matrix compatibility. The point is to make a sovereign, branded, easier deployment path that still works with existing Matrix clients.

## Matrix Gives

- Open chat protocol.
- Client interoperability.
- End-to-end encryption support in compatible clients.
- Room model.
- Identity model.
- Federation option.

## CallChat Adds

- Branded public front door.
- Cleaner setup docs for self-hosters.
- Element Web configuration profile.
- Operator health checks and backup flow.
- Optional AI bot integration path.
- Optional private Shield/ZMath premium layer.

## Why Keep Matrix/Element

Building a full secure messenger from scratch is expensive and risky. Matrix and Element already solve a large amount of the protocol and client problem. CallChat can focus on product, deployment, private features, and user experience.
