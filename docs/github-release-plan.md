# GitHub Release Plan

Suggested first public release:

## v0.1.0

- Public README.
- Synapse/PostgreSQL templates.
- Apache/Nginx examples.
- Website template.
- Element Web config sample.
- CallChat theme CSS.
- DNS/discovery docs.
- Admin guide.
- Backup template.
- Healthcheck script.
- Shield public boundary docs.
- AI bot integration notes.

## Release Assets

Do not upload production backups or private binaries.

Safe assets:

- Source zip generated by GitHub.
- Optional public setup zip built by `scripts/generate-release-zip.ps1`.
- Checksums for public files only.

## Tagging

Use semantic version tags:

```bash
git tag v0.1.0
git push origin v0.1.0
```
