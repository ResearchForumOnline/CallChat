# Self-Hosting Checklist

## Before DNS

- Choose final server name.
- Choose Matrix API hostname.
- Confirm server resources.
- Prepare backups.
- Decide registration policy.

## Before Launch

- HTTPS works.
- `.well-known` files return valid JSON.
- Synapse `/versions` endpoint works.
- First admin account created.
- Public registration closed unless deliberately enabled.
- Element Web login tested.
- Mobile Element login tested.
- Media upload tested.
- Device verification tested.
- TURN calls tested.
- Abuse contact route published.

## Before Public Promotion

- Security policy published.
- Privacy policy adapted.
- Terms adapted.
- Backups tested.
- Admin password stored safely.
- No secrets in the public repo.
- No private Shield code in public assets.
