# Contributing

Thank you for helping CallChat Community become easier to self-host.

## Good Contributions

- Clearer install docs.
- Safer defaults.
- Better health checks.
- Apache/Nginx examples.
- Element Web theme improvements.
- Accessibility fixes.
- Matrix client setup notes.
- Backup/restore improvements.
- Tests for public scripts and templates.

## Boundaries

Do not submit:

- Secrets.
- Real production configs.
- Account passwords.
- Private Shield/ZMath implementation details.
- Copied proprietary code.
- Forked upstream code with missing license notices.

## Pull Request Checklist

- No real credentials or tokens.
- No private keys or signing keys.
- Docs explain how to roll back operational changes.
- Public examples use placeholders.
- Third-party attribution remains intact.
- Shell scripts pass `shellcheck` when possible.
