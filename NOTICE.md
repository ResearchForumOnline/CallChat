# Notice

CallChat Community is a deployment kit and product shell for Matrix-compatible chat. It is not the Matrix protocol, Synapse, or Element Web.

## Third-Party Projects

This repository may reference or configure:

- Matrix protocol and discovery files.
- Synapse homeserver.
- Element Web.
- Docker and Docker Compose.
- PostgreSQL.
- coturn/TURN.
- Optional Matrix SDKs or bot libraries.

Those projects keep their own licenses, trademarks, security policies, and release processes.

## Redistribution Guidance

The safest public distribution model is:

- Keep CallChat source, docs, themes, and templates in this repository.
- Download or install upstream Synapse and Element Web through their official release channels.
- Keep upstream license notices intact.
- Do not remove attribution from upstream clients.
- Do not distribute a closed-source fork of copyleft code without a licensing review.

## Shield Boundary

CallChat Shield / ZMath premium implementation code is not included here. Public docs may describe behaviour and integration points, but private source, entitlement internals, and deployment secrets must stay outside public repositories.
